# E-Commerce Sales Dashboard

An interactive business intelligence dashboard for analyzing e-commerce sales data, identifying trends, and making data-driven decisions.

## Features

- **Key Metrics Overview**: Total revenue, orders, average order value, and conversion rate
- **Sales Trend Analysis**: Interactive line chart showing daily sales performance
- **Category Performance**: Bar chart displaying revenue distribution across product categories
- **Top Products**: Detailed table of best-selling products with rankings
- **Date Range Filtering**: Flexible time period selection (7/30/90 days, yearly)

## Skills & Technologies

- Data visualization with Chart.js
- Interactive dashboards
- Business analytics
- React components
- Responsive design with Tailwind CSS

## Data Insights

The dashboard provides actionable insights for:
- Identifying best-selling products
- Understanding sales trends over time
- Analyzing category performance
- Tracking key business metrics
- Supporting strategic decision-making
